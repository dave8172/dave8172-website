## Overview

We can extract specific data from google sheets, using Apps Script.

We use a simple button, which can also be set in the Menu, which runs the extraction script. This script <RandomColor>filters</RandomColor> the data using our criteria and pastes the result into a new sheet.
Using 1-click button, we can extract data from google sheets using any criteria.

## Google Apps Script <RandomColor>Logic</RandomColor>

The Apps Script:
- Fetches the Google Sheet, the source and destination sheets
- Gets the user input for setting the criteria
- Filters the source data based on criteria
- Pastes the filtered results into the destination sheet
- This Script is linked to a button in Google Sheets, and runs when the button is clicked
